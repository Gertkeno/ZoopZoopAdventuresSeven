## Building

Made with [Zig](https://ziglang.org) 0.14.0 for [Wasm4](https://wasm4.org)

To build and run, use these commands

```shell
zig build -Doptimize=ReleaseSmall
w4 run zig-out/lib/cart.wasm
```
